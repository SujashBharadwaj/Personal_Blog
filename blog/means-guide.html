A practical guide to means in statistics

Published: Sep 1, 2025 · Reading time: ~7 min

Different “means” answer different questions. For positive numbers, a good rule of thumb is: Harmonic ≤ Geometric ≤ Arithmetic ≤ RMS ≤ Contraharmonic.

Example dataset

Use x = [1, 2, 8].

Arithmetic mean (AM)

Average of values. Formula: (1/n) Σ xi. Example: (1+2+8)/3 = 3.6667. Use for additive quantities.

Geometric mean (GM)

Typical multiplicative change. Formula: (∏ xi)1/n. Example: (1·2·8)1/3 = 2.5198. Requires x>0.

Harmonic mean (HM)

Average of rates. Formula: n / Σ(1/xi). Example: 3 / (1 + 0.5 + 0.125) = 1.8462. Requires x>0.

RMS (quadratic mean)

Emphasizes large values. Formula: √[(1/n) Σ xi2]. Example: √(23) = 4.7958.

Contraharmonic mean

Even more weight on big values. Formula: (Σ xi2) / (Σ xi). Example: 69/11 = 6.2727 (nonnegative data).

Weighted and trimmed means

  • Weighted: Σ wixi / Σ wi — use when some observations matter more.
  • Trimmed: drop extremes, then take AM — more robust against outliers.

Power-mean family (generalized mean)

Mp(x) = [(1/n) Σ xip]1/p. p = −1 → HM, p→0 → GM, p = 1 → AM, p = 2 → RMS.

When to use what

SituationPickWhy
Default reportingAMStraight average of values
Growth/ratiosGMCaptures multiplicative effects
Speeds/efficienciesHMAverages “per unit” correctly
Magnitudes/energyRMSSquares stress large values
Outliers presentTrimmed meanMore robust than AM
Different importancesWeighted meanRespects weights

Always name the mean you used and why.
