A practical guide to means in statistics

Published: Sep 1, 2025 • Reading time: 7–9 min

TL;DR: For positive data, the classic order is Harmonic ≤ Geometric ≤ Arithmetic ≤ Quadratic (RMS) ≤ Contraharmonic. Each mean answers a different “average” question.

Dataset we’ll use

Let’s use x = [1, 2, 8] (all positive—important for some means).

Arithmetic Mean (AM)

“Evenly share the total.”

Formula: \( \bar{x} = \frac{1}{n} \sum x_i \)

Example: (1 + 2 + 8) / 3 = 11 / 3 ≈ 3.6667

Use when: Additive quantities; typical default. Not robust to outliers.

Geometric Mean (GM)

“Typical multiplicative growth.” (Requires all values > 0.)

Formula: \( \left(\prod x_i\right)^{1/n} \)

Example: (1 × 2 × 8)^(1/3) = 16^(1/3) ≈ 2.5198

Use when: Growth rates, ratios, normalized scores.

Harmonic Mean (HM)

“Average of rates.” (Requires all values > 0.)

Formula: \( \displaystyle HM = \frac{n}{\sum (1/x_i)} \)

Example: 3 / (1/1 + 1/2 + 1/8) = 3 / 1.625 = 1.8462

Use when: Speeds, efficiencies, “items per unit time”. Penalizes small values strongly.

Quadratic Mean (RMS)

“Energy/magnitude average.”

Formula: \( \displaystyle \sqrt{\frac{1}{n}\sum x_i^2} \)

Example: sqrt((1² + 2² + 8²) / 3) = sqrt(69 / 3) = sqrt(23) ≈ 4.7958

Use when: Signals, errors, distances; emphasizes large values.

Contraharmonic Mean (C)

“Intensity-weighted average.” (Requires nonnegative values.)

Formula: \( \displaystyle C = \frac{\sum x_i^2}{\sum x_i} \)

Example: (1 + 4 + 64) / (1 + 2 + 8) = 69 / 11 ≈ 6.2727

Use when: You want to reward larger values even more than RMS (e.g., bright pixels).

Weighted Mean

“Some observations matter more.”

Formula: \( \displaystyle \bar{x}_w = \frac{\sum w_i x_i}{\sum w_i} \) (usually \( \sum w_i = 1 \))

Example: values [10,20,30], weights [0.2,0.3,0.5] ⇒ 10·0.2 + 20·0.3 + 30·0.5 = 23

Trimmed Mean

“Average without the extremes.”

Remove a fixed % from both ends after sorting, then take AM.

Example: [5, 6, 7, 100] with 25% trim ⇒ drop 5 and 100 ⇒ mean(6,7) = 6.5

Use when: You want robustness to outliers but still keep a mean.

The Power Mean family (Generalized Mean)

\( \displaystyle M_p(x) = \left(\frac{1}{n}\sum x_i^p\right)^{1/p} \). For positive numbers:

  • p → −∞ gives min
  • p = −1 → Harmonic
  • p → 0 → Geometric (limit)
  • p = 1 → Arithmetic
  • p = 2 → Quadratic (RMS)
  • p → +∞ gives max

Order (for positive data): HM ≤ GM ≤ AM ≤ RMS ≤ Contraharmonic.

Which mean should I use?

SituationPreferWhy
Additive totals / default reportingArithmeticStraight average of values
Multiplicative effects (growth, ratios)GeometricCaptures typical proportional change
Rates (e.g., km/h across segments)HarmonicCorrectly averages “per unit” quantities
Signal magnitudes / energyRMSSquares emphasize larger magnitudes
Outliers present, keep meanTrimmed meanMore robust than AM
Different importancesWeighted meanReflects weights (credits, sample sizes)

Key cautions

  • GM & HM need positive values. Zero/negative breaks them.
  • AM & RMS are outlier-sensitive. Consider trimming or median-based summaries.
  • Always state which mean you used and why.
